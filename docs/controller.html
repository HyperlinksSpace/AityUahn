One-line install (recommended)

irm https://raw.githubusercontent.com/HyperlinksSpace/AityUahn/main/scripts/install.ps1 | iex
curl -LsSf https://raw.githubusercontent.com/HyperlinksSpace/AityUahn/main/scripts/install.sh | sh

Installs to %USERPROFILE%\AityUahn on Windows. Then run serve.bat or aityuahn serve.

Manual install

  1. Extract the ZIP or git clone the repo, then cd AityUahn
  2. python -m venv .venv — activate: source .venv/Scripts/activate (Git Bash) or .\.venv\Scripts\Activate.ps1 (PowerShell)
  3. pip install -e ".[dev]" — then aityuahn init to confirm the CLI
  4. cp config/forge.example.yaml forge.yaml and cp .env.example .env — set workspace_root and API keys (keys optional for sign-up only)
  5. aityuahn serve — API at http://127.0.0.1:8765 (try aityuahn serve --demo for sample data)
  6. Verify: open http://127.0.0.1:8765/api/health or run curl http://127.0.0.1:8765/api/health

After connecting, complete sign-in or registration in the dialog. Team features require an account. See docs for a full UI test checklist.
